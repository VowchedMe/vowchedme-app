# VowchedMe.com (starter)

A dark, premium marketing site + auth UI stubs for VowchedMe.

## Run locally
1. Install deps:
```bash
npm install
```

2. Start dev server:
```bash
npm run dev
```

Open http://localhost:3000

## Pages included
- `/` Home
- `/job-seekers`
- `/employers`
- `/employers/dashboard` (preview)
- `/profile/demo` (profile with 3 pinned Vowches)
- `/pricing`
- `/faq`
- `/about`
- `/contact`
- `/waitlist`
- `/auth/sign-in`
- `/auth/sign-up`
